## ğŸ”§ Troubleshooting

### Common Issues:

1. **MongoDB Connection Failed**
   - Solution: The app automatically falls back to in-memory storage
   - No action required for testing

2. **API Key Errors**
   - Solution: Set up environment variables in .env file
   - AI features require Hugging Face API token

3. **Port Already in Use**
   - Solution: Change port in app.py or close other applications

4. **Missing Dependencies**
   - Solution: Run `pip install -r requirements.txt` again

5. **Virtual Environment Issues**
   - Solution: Delete venv folder and recreate it

### Getting Help:

1. Check the terminal output for error messages
2. Verify all dependencies are installed
3. Ensure API keys are properly configured
4. Check if MongoDB is running (optional)
